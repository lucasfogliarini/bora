# Bora

Bora é uma linguagem computacional com raiz lá do tempo de Jesus Cristo, Sócrates e Euclides, mas no nosso sotaque bagual, de pé no chão e sem balaca.

## Atribuindo
```csharp
= //Bota um valor(elemento ou número) no lugar. Nada a ver com Comparando se é igual.
```
## Desengronhando e Engronhando

```csharp
+     //Desengronhar pouco
-     //Engronhar pouco
*     //Desengronhar muito
/     //Engronhar muito
```
## Comparando
```csharp
== //Isso sim é comparar se é igual Vivente! E não é o mesmo que atribuir.
!=    //Isso é completamente oposto de '=='.
>     //Maior que um trosso
<     //Menor que um trosso
>=    //Maior ou igual que um trosso
<=    //Menor ou igual que um trosso
```
